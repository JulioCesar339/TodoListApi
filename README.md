# Todo API con .NET 8 + PostgreSQL + Docker

API REST completa de tareas (CRUD) con:
- .NET 8
- Entity Framework Core 8
- PostgreSQL 16
- Docker + Docker Compose
- Migraciones automáticas al iniciar

¡Se levanta con un solo comando en cualquier máquina!

## Levantar el proyecto (10 segundos)

```bash
git clone https://github.com/JulioCesar339/TodoListApi.git
cd TodoListApi
docker compose up -d --build
